Menüzeile

Die Menüzeile hat die folgenden untermenüs

  • Datei
    • Zeitserie ausdrucken mit Drucken
    • Zeitserie als Bild speichern mit Bild speichern Hier hat man die Wahl zwischen XPM/BMP/PNG og EPS. Die Standarteinstellung ist PNG. Alle formate ausser EPS sind screendumps. Nur EPS ist dynamisch.
    • Man kann seinen eigenen Positionsfilter generieren mit Filter ändern
    • Abschließen beendet T-series
  • Einstellungen
    • Man kann die aktuellen Einstellungen wie Fenstergrösse/ -positionierung, Ort, Typ, Sprache Model mit Einstellungen speichern abspeichern. Die gespeicherten Werte werden verwendet wenn man T-series startet oder Einstellungen zurücksetzen wählt.
    • Wenn Beim Abschliessen speichern gewählt ist, werden die Werte automatisch gespeichert wenn man T-series beendet.
    • Ortsnamen anzeigen (DIANA) schreibt alle ortsnamen des aktuellen Datensatzes auf die DIANA-karte. Kann bei größeren Listen störend wirken.
    • Aktueller Ortsname (DIANA) Schreibt den Namen des angewählten Ortes in DIANA (empfohlen)
    • Zeitmarkierung Eine kleine graue Linie, die die aktuelle Zeit im Diagram repräsentiert. Die Linie ist an die interne Uhr angeschlossen. Zusätzlich erhält man beim Anschluß an DIANA eine rote Linie. Diese represäntiert die aktuelle Zeit der angezeigten Modellfelder in DIANA.
    • Sprachen Auswahl der Menüsprache. Diagramsprache muß via style-file gewählt werden.
  • Hilfe
    • Benutzerhandbuch zeigt dieses Dokument
    • ÜberVersionsnummer etc.

An DIANA anschließen


T-series kan an eine diana-instanz angeschlossen werden wenn diese dafür geöffnet wurde.
Die Verbindung entsteht durch einen externen (socket-basierten) Server. Der Server wird automatisch von T-series oder DIANA gestartet. Die Verbingung wird hergestellt wenn in beiden Applikationen der knopf angewählt wurde.

Nach der Verbindung geschieht das Folgende:

  • T-series schickt eine komplette Positionsliste an DIANA. Diese Positionen erscheinen auf der DIANA-Karte als ikonen
  • Wenn man mit dem Mauszeiger eine der ikonen anzeigt, schickt DIANA den angewählten Ortsnamen zurück an T-series, welches dann die entsprechende Zeitserie anzeigt.
  • Durch das anklicken von Läßt man T-series einen Zeiger an DIANA senden, der die aktuelle Position hervorhebt.


Positionsfilter

Man kann die anzahl der angezeigten Positionen begrenzen. Hierzu kann man den Filter anwählen. Der Filter har eine Standarteinstellung die vom Benutzer geändert werden kann. Der geänderte Filter kann persönlich abgespeichert werden. Hierzu verwendet man den Positionsfilterdialog der über Datei / Filter ändern gestartet wird.

Der Dialog hat zwei Positionslisten, einen der alle Positionen enthält (links) und einen der den persönlichen Filter repräsentiert (rechts). Man kann velge Namen im Filter anwählen und ( oder die DEL-taste ) drücken um eine Position aus dem Filter au entfernen. Drückt man (oder die SPACE-Taste) Wird die Positition, die in der Haupliste angewählt ist in den Filter kopiert.
Der Filter kan auf den Standart zurückgesetzt werden indem man drückt.
